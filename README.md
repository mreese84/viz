# Cycling Viz Project

## About
An experiment in turning cycling data into bespoke artwork.

## Installation

### Prerequisites

* Node.js 6.2.2+
* NPM 3.9.5+
* Gulp 3.9.1+

### Initial setup

Run

    npm install


### Running

Start gulp using `gulp watch`.

    gulp watch

A browser window should open at `http://localhost:9000`.
